AUIGrid 2.1 Samples

■ 테마는 CSS 파일 교체만으로 가능합니다.
예를 들어 다크 테마로 바꾸고 싶다면 해당 소스에서 AUIGrid_style.css 파일 대신 AUIGrid_dark_style.css 로 수정해 주세요

■ 엑셀, PDF Export 관련 데모는 반드시 exportURL 속성을 서버 환경에 맞게 고치고 실행해주세요.
export_server_samples 폴더에 PHP, JSP, ASP 용 소스가 있습니다.(ROOT 폴더 아래에 있음)

■ PDF 저장을 위한 JS 파일과 글꼴(Font) 파일은 pdfkit 폴더에 위치합니다. PDF 저장을 원한다면 반드시 AUIGrid.pdfkit.js 파일을 AUIGrid.js 파일 하단에 추가 하십시오.

■ 쇼케이스 "사원 관리"와 "DB 그리드"의 서버 사이드 스크립트(실 DB Select, Update, Delete 하는 PHP 소스)는 포함되지 않았습니다.

필요하신 분은 aui@auisoft.net 으로 연락주시면 보내드리겠습니다.

■ "unauthorized hosts for AUIGrid" 메세지 출력에 대한 대응

  • 평가판 사용자 : 웹 서버에 올리지 않고 바로 윈도우 탐색기에서 index.html 을 실행한 경우입니다.
    반드시 웹 서버에 올린 후 http://... 또는 https:// 와 같은 URL 경로로 접근하십시오.(C://.. 과 같은 접근 허용은 별도의 라이센스 필요)
  • 정품 사용자 : 구매한 정품 라이센스의 호스트가 아닌 곳에서 AUIGrid 를 실행하고 있는 경우입니다.
